HTML es el lenguaje base para estructurar cualquier página web. En este artículo, te presentamos una guía completa con 100 etiquetas, organizadas por categorías, que te ayudarán a crear documentos web semánticos, accesibles y optimizados para SEO.

1. Etiquetas para la base del DOM

  • <!DOCTYPE>: Define el tipo de documento HTML.
  • <html>: Contenedor principal del documento. (Atributo: lang)
  • <head>: Contiene metadatos y enlaces a recursos externos.
  • <body>: Define el contenido visible de la página.
  • <title>: Especifica el título del documento.
  • <meta>: Proporciona metadatos, como charset, descripción y keywords.
  • <base>: Define la URL base para enlaces relativos. (Atributos: href, target)
  • <link>: Conecta el documento con hojas de estilo externas.
  • <style>: Incluye estilos CSS en línea.
  • <details> y <summary>: Crea secciones desplegables.
  • : Comentarios en el código HTML.

2. Etiquetas para dividir secciones

  • <div>: Contenedor genérico para agrupar contenido.
  • <section>: Sección temática del contenido.
  • <article>: Contenido independiente y autocontenible.
  • <aside>: Contenido relacionado o complementario.
  • <header>: Encabezado de una sección o documento.
  • <footer>: Pie de página o cierre de una sección.
  • <nav>: Conjunto de enlaces de navegación.
  • <main>: Contenido principal de la página.

3. Etiquetas para crear textos

  • <p>: Define un párrafo.
  • <h1> a <h6>: Encabezados de distintos niveles.
  • <hgroup>: Agrupa encabezados.
  • <a>: Crea un enlace o hipervínculo. (Atributos: href, target, etc.)
  • <strong>: Texto con énfasis fuerte (negrita).
  • <em>: Texto enfatizado (cursiva).
  • <u>: Texto subrayado.
  • <del>: Texto tachado.
  • <sup>: Texto sobrescrito.
  • <sub>: Texto subíndice.
  • <blockquote>: Cita extensa.
  • <q>: Cita corta. (Atributo: cite)
  • <pre>: Texto preformateado.
  • <abbr>: Abreviatura. (Atributo: title)
  • <address>: Información de contacto.
  • <cite>: Título de una obra o fuente.
  • <bdo>: Cambia la dirección del texto. (Atributo: dir)
  • <mark>: Texto resaltado.
  • <span>: Contenedor en línea para aplicar estilos.
  • <small>: Texto pequeño.
  • <time>: Fecha u hora. (Atributo: datetime)
  • <var>: Variable o nombre de una variable.
  • <dfn>: Término definido.
  • <bdi>: Aísla un fragmento de texto.
  • <br>: Salto de línea.
  • <wbr>: Punto de posible quiebre de línea.
  • <hr>: Línea horizontal para separar contenido.

4. Etiquetas para listas

  • <ul>: Lista desordenada. (Atributos: type, start, reversed)
  • <ol>: Lista ordenada.
  • <li>: Elemento de una lista.
  • <dl>: Lista de definiciones.
  • <dt>: Término dentro de una lista de definiciones.
  • <dd>: Descripción o definición de un término.

5. Etiquetas para tablas

  • <table>: Crea una tabla. (Atributos: border, cellpadding, cellspacing, width)
  • <caption>: Título de la tabla.
  • <th>: Celda de encabezado. (Atributos: colspan, rowspan)
  • <tr>: Fila de la tabla.
  • <td>: Celda de la tabla. (Atributos: colspan, rowspan)
  • <thead>: Agrupa el encabezado de la tabla.
  • <tbody>: Agrupa el cuerpo de la tabla.
  • <tfoot>: Agrupa el pie de la tabla.
  • <col>: Define propiedades para columnas.
  • <colgroup>: Agrupa columnas.

6. Etiquetas para multimedia y scripts

  • <script>: Inserta un script (generalmente JavaScript). (Atributos: src, type, async, defer, crossorigin)
  • <noscript>: Contenido alternativo para navegadores sin soporte de scripts.
  • <canvas>: Área para gráficos y dibujos dinámicos. (Atributos: width, height)
  • <audio>: Archivo de sonido o música. (Atributos: src, controls, etc.)
  • <video>: Archivo de video. (Atributos: src, width, height, controls, etc.)
  • <source>: Define múltiples recursos para audio o video.
  • <track>: Subtítulos o pistas para elementos multimedia.
  • <iframe>: Inserta un marco en línea para cargar otra página. (Atributos: src, width, height, etc.)
  • <embed>: Inserta contenido integrado, como plugins.
  • <object>: Inserta un objeto incrustado. (Atributos: data, type, width, height, etc.)
  • <param>: Define parámetros para un objeto.
  • <map>: Define un mapa de imagen. (Atributo: name)
  • <area>: Define una región dentro de un mapa de imagen. (Atributos: shape, coords, href, alt)
  • <progress>: Barra de progreso. (Atributos: value, max)
  • <meter>: Muestra un valor dentro de un rango conocido. (Atributos: value, min, max, etc.)
  • <code>: Fragmento de código.
  • <kbd>: Texto representativo de una entrada de teclado.
  • <samp>: Muestra la salida de un programa.
  • <var>: Representa una variable.

7. Etiquetas para formularios

  • <form>: Crea un formulario HTML. (Atributos: action, method, target)
  • <input>: Campo de entrada. (Atributos: type, name, value, placeholder, etc.)
  • <textarea>: Área para texto. (Atributos: name, rows, cols, etc.)
  • <select>: Lista desplegable. (Atributos: name, multiple, size, etc.)
  • <option>: Opción dentro de una lista. (Atributos: value, selected, disabled)
  • <label>: Etiqueta para un elemento de formulario. (Atributo: for)
  • <fieldset>: Agrupa elementos relacionados de un formulario.
  • <legend>: Título para un <fieldset>.
  • <button>: Botón en un formulario. (Atributos: type, etc.)
  • <datalist>: Lista de opciones predefinidas para un campo de entrada.
  • <optgroup>: Agrupa opciones dentro de un <select>. (Atributo: label)
  • <input type="radio">: Botón de opción.
  • <input type="checkbox">: Casilla de verificación.
  • <input type="submit">: Botón para enviar el formulario.
  • <input type="reset">: Botón para reiniciar el formulario.
  • <input type="image">: Imagen que actúa como botón de envío.
  • <input type="hidden">: Campo oculto en el formulario.

8. Conclusión

Esta guía abarca las 100 etiquetas esenciales de HTML, organizadas en categorías para facilitar su estudio y aplicación. El uso correcto de estas etiquetas mejora la semántica, la accesibilidad y el SEO de tus proyectos, permitiéndote crear páginas web robustas y bien estructuradas.
¡Experimenta y potencia tus webs con todo el poder oculto de HTML!
