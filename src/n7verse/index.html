Sobre mí

Soy desarrolladora Frontend resolutiva, apasionada por crear interfaces atractivas, funcionales y centradas en la experiencia de usuario. Disfruto especialmente diseñando con Tailwind CSS y dando vida a ideas con JavaScript, HTML, CSS y frameworks como Angular.

Me gusta cuidar los detalles visuales, pero también sé construir mis propios backends cuando hace falta, integrando APIs, bases de datos o soluciones personalizadas con Node.js, MongoDB, PHP o Firebase.

Trabajo de forma práctica, creativa y autónoma, con el objetivo de desarrollar webs modernas, responsivas y bien optimizadas para buscadores. Siempre estoy aprendiendo y explorando nuevas herramientas que me permitan entregar resultados que realmente conecten.
